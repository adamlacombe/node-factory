# Types

## `FactoryGenerator`

```typescript
type FactoryGenerator = (fake: Faker.FakerStatic) => DataObject<T>;
```

> For more information on Faker.FakerStatic type refer to https://www.npmjs.com/package/@types/faker

## `Factory`

```typescript
interface Factory<T = any> {
  make: MakeMethod<T>;
  create: CreateMethod<T>;
  only: (keys: keyof T | Array<keyof T>, overrides?: DataObject | FactoryGenerator) => Partial<T>;
  seed: (value: number) => Factory<T>;
  state: (name: string, stateValues: DataObject | FactoryGenerator) => void;
  onInsert(func: DatabaseConfig<T>['insert']): Factory<T>;
  onHydrate(func: DatabaseConfig<T>['hydrate']): Factory<T>;
}
```
