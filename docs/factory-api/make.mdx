# `Factory<T, S>.make`

## Description

This method is used to generate data objects withouth persisting to a database. As a rule of thumb, if you're **NOT** testing a database, use the `make` method. Otherwise, use the create method.

## Type

```typescript
interface MakeMethod<T> {
  (): T;
  (overrides: DataObject): T;
  (count: number): T[];
  (count: number, overrides: DataObject): T[];
}
```

> Where `T` is the type passed to the factory function.

### args

- `count `<span>`(Number [optional])`</span>: This argument defines how many objects should be generated `(default: 1)`
- `overrides `<Link to="/types#dataobject">`(DataObject [optional])`</Link>: These are value which should be overridden on the generated data `(default: {})`

### return

- `data `<Link to="/types#dataobject">`(DataObject | DataObject[])`</Link>: A data object or an array containing data objects generated by your generator function
