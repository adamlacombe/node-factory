# `Factory<T, S>.create`

## Description

<span>This method is used to generate a data object and insert it into a database. To do that, it uses the database configuration defined using the </span>
<Link to="/api/onInsert">`onInsert`</Link>
<span> method. As a rule of thumb, if you <strong>ARE</strong> testing a database, use the create method, otherwise use the make method.</span>

## Type

```typescript
interface CreateMethod<T> {
  (): Promise<T>;
  (overrides: DataObject): Promise<T>;
  (count: number): Promise<T[]>;
  (count: number, overrides: DataObject): Promise<T[]>;
}
```

> Where `T` is the type passed to the factory function.

### args

- `count (Number [optional])`: This argument defines how many objects should be generated (default: 1)
- `overrides `<Link to="/types#dataobject">`(DataObject [optional])`</Link>: These are value which should be overridden on the generated data `(default: {})`

### return

- `data (Promise<`<Link to="/types#dataobject">`DataObject`</Link>` | `<Link to="/types#dataobject">`DataObject[]`</Link> `>)`: Data object or list of data objects generated by your generator function
