# `Factory.only(fields[, overrides])`

### args

- `fields `<span>`(String | Array)`</span>: This argument defines the fields that should be returned.

> - If a `String`, returns an object containing only that key <br />
> - If an `Array`, returns an object containing only the keys contained in the array

- `overrides `<Link to="/api/types#idataobject">`(IDataObject [optional])`</Link>: These are value which should be overridden on the generated data `(default: {})`

### return

- `data `<Link to="/api/types#idataobject">`(IDataObject)`</Link>: Data object generated by your generator function

## Description

This method is used to generate a data object containing only the keys specified. This could be particularly useful for updating state.

## Example

```js
UserFactory.only('email')
```

<FormatJson parse={() => UserFactory.only('email')} />

```js
UserFactory.only(['email', 'name'])
```

<FormatJson parse={() => UserFactory.only(['email', 'name'])} />

```js
UserFactory.only(['id', 'email'], { id: '123' })
```

<FormatJson parse={() => UserFactory.only(['id', 'email'], { id: '123' })} />
